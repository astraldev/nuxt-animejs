The `useLayout` composable helps with layout animations.

## Type Definition

```ts
function useLayout(
  target: AnimeLayoutTargets,
  options: AutoLayoutParams
): {
  properties: ComputedRef<NonFunctionProperties<AutoLayout>>,
  record: () => void,
  revert: () => void,
  animate: (params?: LayoutAnimationParams) => void
}
```

### Arguments

::field-group
  ::field{name="target" type="AnimeLayoutTargets" required}
  The wrapper element or selector for the layout transition.
  ::

  ::field{name="options" type="AutoLayoutParams"}
  Auto layout parameters. See [AnimeJS Layout documentation](https://animejs.com/documentation/#layout).
  ::
::

## Sample usage

::code-group
```ts [script]
const items = ref([])
const { record, animate } = useLayout('.layout-wrapper', {
  duration: 500,
  ease: 'inOutQuad'
})

// Trigger layout animation after DOM changes
const addItem = () => {
  record()
  items.value.push({ name: items.value.length + 1 })
  nextTick(animate)
}
```

```vue [template]
<template>
  <div class="layout-wrapper">
    <div v-for="item in items" :key="item.id" class="layout-item">
      {{ item.name }}
    </div>
  </div>
</template>
```
::

## Return Value

Returns an object with helper methods to control the layout animation.

::collapsible

| Property | Type | Description |
| :--- | :--- | :--- |
| `properties` | `ComputedRef<object>`{lang="ts-type"} | The current layout properties. |
| `record` | `() => void`{lang="ts-type"} | Record the current layout state. |
| `revert` | `() => void`{lang="ts-type"} | Revert the layout to its original state. |
| `animate` | `(params?: LayoutAnimationParams) => void`{lang="ts-type"} | Animate from the recorded state to the current state. |

::

## API

### Types

```ts
type AnimeLayoutTargets = DOMTargetSelector | MaybeElementRef<HTMLElement | VueInstance> | null | undefined
```
